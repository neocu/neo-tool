### Usage

##### noe page
根据模版创建新页面
```
$ neo p index
```

创建新组件
```
$ neo c modal
```

查看帮助
```
$ neo --help
```

### Development

1. 安装依赖包
```
npm install
```

2. 编译ES6
```
webpack -d
```
